# lab6
Лабораторная работа 6. Построение и визуализация трехмерных объектов. Реализация трехмерных преобразований. Построение простейших проекций.
Цель работы: 
Закрепить теоретический материал и практическое освоение основных методов и алгоритмов трёхмерной визуализации, а также научиться реализовывать базовые преобразования и проекции трёхмерных объектов.

Задачи:
Написать приложение/веб‑приложение, в котором:
- формируется и визуализируется заданный трёхмерный объект (каркасная модель). В качестве трёхмерного объекта берётся трёхмерное изображение первой буквы фамилии студента - G.
- реализовать трёхмерные преобразования заданного объекта: масштабирование, перенос, вращение вокруг произвольной оси. При выполнении преобразований необходимо выводить итоговую матрицу преобразования.
- реализовать построение трёх ортографических проекций заданного объекта на координатные плоскости: Oxy, Oxz, Oyz.

Средства разработки: 

- Языки: HTML, CSS, JavaScript
- Графика: Canvas API
- Контейнеризация: Docker

Возможности приложения:

- Построение каркасной модели буквы.
- Масштабирование, перенос, вращение вокруг произвольной оси.
- Отображение трёх ортографических проекций (Oxy, Oxz, Oyz).
- Визуализация координатных осей и системы координат.
- Вывод матрицы преобразований.
- Запуск в Docker‑контейнере для удобного распространения.

Docker:
docker run -d -p 8080:80 tatttti/gproject:latest
Then open: http://localhost:8080

    
Вывод: В ходе лабораторной работы разработано веб‑приложение для построения и визуализации трёхмерных объектов. Реализованы трёхмерные преобразования (масштабирование, перенос, вращение вокруг произвольной оси) и построение ортографических проекций на плоскости Oxy, Oxz, Oyz. Приложение снабжено графическим интерфейсом с системой координат и выводом матрицы преобразований. Для удобства демонстрации и распространения проект упакован в Docker‑контейнер и может быть запущен на любом компьютере.
