# lab5
Лабораторная работа 5. Алгоритмы отсечения отрезков и многоугольнико
Цель работы: 
Закрепить теоретические знания и практическое освоение оосновных методов и алгоритмов отсечения отрезков и многоугольников:
- алгоритм средней точки
- отсечение выпуклого многоугольника произвольным выпуклым отсекателем

Задачи:
Написать приложение/веб-приложение, иллюстрирующее работу алгоритмов отсечения отрезков и многоугольников.

Средства разработки: 

- Языки: HTML, CSS, JavaScript
- Графика: Canvas API

Возможности приложения:

- Визуализация алгоритмов отсечения 
- Масштабирование и панорамирование сцены
- Отображение координатной сетки и осей
- Подписи координат и элементов
- Переключение между алгоритмами отсечения
- Вывод информации об отсечённых частях
- Загрузка входных данных через файл или интерфейс

Формат входных данных для алгоритма средней точки:
n
X1_1 Y1_1 X2_1 Y2_1
X1_2 Y1_2 X2_2 Y2_2
...
X1_n Y1_n X2_n Y2_n
Xmin Ymin Xmax Ymax

Формат входных данных для отсечение выпуклого многоугольника произвольным выпуклым отсекателем:
m
X1 Y1
X2 Y2
...
Xm Ym
k
CX1 CY1
CX2 CY2
...
CXk CYk

Docker:
docker run -d -p 8080:80 tatttti/clipping-algorithms:latest
Then open: http://localhost:8080

    
Вывод: В ходе лабораторной работы разработано веб-приложение для визуализации алгоритмов отсечения отрезков и многоугольников. Реализованы алгоритм средней точки и отсечение выпуклого многоугольника произвольным выпуклым отсекателем с интерактивным интерфейсом, поддержкой загрузки данных и профессиональным оформлением. Приложение упаковано в Docker-контейнер для удобства распространения и использования.
